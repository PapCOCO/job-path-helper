# 职途清单 - Job Path Helper

大学生求职 JD 解析与能力补齐助手。

## 项目简介

职途清单是一个面向理工科大学生的求职辅助工具，包含三个核心功能：

1. **本地规则分析**：基于预定义的规则库，解析 JD 并进行岗位类型识别、技能匹配、差距分析，提供 7 天/30 天行动清单、简历包装建议和面试表达建议。

2. **大模型增强分析**：通过 OpenRouter 调用百度 CoBuddy 免费模型（`baidu/cobuddy:free`），在本地规则分析的基础上提供更自然的总结、更具体的简历润色、更贴合学生背景的面试表达、更可执行的补强建议。

3. **JD 截图识别**：通过 OpenRouter 调用百度 OCR 免费模型（`baidu/qianfan-ocr-fast:free`），识别上传的 JD 截图中的文字，自动填入 JD 输入框。

## 快速开始

### 1. 安装依赖

```bash
npm install
```

### 2. 配置环境变量

复制 `.env.example` 为 `.env`，然后填入你的 OpenRouter API Key：

```bash
cp .env.example .env
# 编辑 .env 文件，将 OPENROUTER_API_KEY 替换为你的实际 Key
```

你可以在 https://openrouter.ai/ 免费注册并获取 API Key。

### 3. 启动服务器

```bash
npm start
```

### 4. 访问应用

在浏览器中打开 http://localhost:3000 即可使用。

## 功能说明

### 本地规则分析（核心兜底功能）

- **岗位类型识别**：识别 7 类技术岗位：嵌入式、电气 PLC、硬件 PCB、硬件测试、机器人、后端、运营。
- **技能拆解与匹配**：三层分类：已掌握、有基础、明确缺口。
- **差距去重**：自动合并表达相同意思的差距。
- **岗位推荐**：无 JD 时根据个人背景推荐适合方向。
- **通用能力分析**：简历表达、项目证明、工具使用、面试表达、求职规划五个维度。

### 大模型增强分析（可选增强功能）

- **分析总结**：自然语言总结整体匹配情况。
- **岗位类型复核**：对本地分析结果进行校验。
- **匹配度复核**：重新评估匹配分数和等级。
- **个人优势**：识别并说明个人核心优势，提供简历写法建议。
- **主要能力缺口**：明确列出核心缺口及补齐方案。
- **优先级行动**：按优先级安排行动项，标明时间和预期产出。
- **简历润色建议**：提供前后对比及修改理由。
- **面试表达要点**：模拟面试问题并给出回答提示。
- **注意事项**：特别提醒需要注意的问题。

### JD 截图识别（可选辅助功能）

- **截图上传**：支持上传 PNG、JPG、WebP 格式的 JD 截图。
- **OCR 识别**：调用 `baidu/qianfan-ocr-fast:free` 模型识别截图中的文字。
- **自动填入**：识别结果自动填入 JD 输入框，用户可检查后开始分析。
- **图片大小**：建议 5MB 以内，超过限制会提示压缩。
- **识别失败兜底**：识别失败时提示用户手动粘贴 JD 文本。

## 模型说明

| 功能 | 模型 | 说明 |
|------|------|------|
| 大模型增强分析 | `baidu/cobuddy:free` | 通过 OpenRouter 调用，用于岗位匹配深度分析 |
| JD 截图识别 | `baidu/qianfan-ocr-fast:free` | 通过 OpenRouter 调用，用于识别截图中的文字 |

两个功能共用 `OPENROUTER_API_KEY`，API Key 只配置在后端 `.env` 文件中，不暴露到前端。

## 注意事项

- 请务必保管好你的 API Key，不要将 `.env` 文件提交到公开仓库。
- 如果大模型调用失败，本地规则分析仍然可用。
- 截图识别失败时，可以手动粘贴 JD 文本。
- 使用的是免费模型，可能存在限流或不可用情况。
- 工具主要面向理工科学生，对医学、法学、金融等岗位支持有限。
